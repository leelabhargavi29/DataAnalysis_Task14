Task 14 - Time Series Analysis

Objective:
Analyze the AirPassengers dataset using Time Series Analysis techniques.

Tools Used:
- Python
- Pandas
- Matplotlib
- Statsmodels

Dataset:
AirPassengers Dataset

Steps Performed:
1. Load the dataset.
2. Convert Month column into datetime.
3. Plot the time series.
4. Perform seasonal decomposition.
5. Perform Augmented Dickey-Fuller (ADF) Test.
6. Train ARIMA model.
7. Forecast future passenger values.
8. Visualize actual vs forecast values.

Output:
- Time Series Plot
- Decomposition Plot
- ADF Test Result
- Forecast Plot

Interview Questions:

1. What is Stationarity?
A stationary time series has constant mean, variance and covariance over time.

2. Explain ARIMA Components?
A – Auto Regression (AR): Uses previous values.
I – Integrated (I): Differencing to make data stationary.
M – Moving Average (MA): Uses previous prediction errors.
